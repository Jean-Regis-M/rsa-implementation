üîê RSA Algorithm Overview

RSA (Rivest-Shamir-Adleman) is one of the first practical public-key cryptosystems and is widely used for secure data transmission. It's based on the practical difficulty of factoring the product of two large prime numbers.

Mathematical Foundation

Key Generation:

  1. Choose two distinct large prime numbers p and q
  2. Compute n = p √ó q (modulus)
  3. Compute œÜ(n) = (p-1) √ó (q-1) (Euler's totient function)
  4. Choose public exponent e such that 1 < e < œÜ(n) and gcd(e, œÜ(n)) = 1
  5. Compute private exponent d = e‚Åª¬π mod œÜ(n)

Encryption: c = m·µâ mod n

Decryption: m = c·µà mod n

üíª Code Examples

Basic Usage

# Initialize RSA with 1024-bit keys
rsa = RSA(key_size=1024)

# Generate public and private keys
public_key, private_key = rsa.generate_keys()

# Encrypt a number
original_number = 42
encrypted_number = rsa.encrypt(original_number)
decrypted_number = rsa.decrypt(encrypted_number)

print(f"Original: {original_number}")
print(f"Encrypted: {encrypted_number}")
print(f"Decrypted: {decrypted_number}")

String Encryption

# Encrypt and decrypt strings
message = "Secret Message 123!"
encrypted_chunks = rsa.encrypt_string(message)
decrypted_message = rsa.decrypt_string(encrypted_chunks)

print(f"Original: '{message}'")
print(f"Decrypted: '{decrypted_message}'")
print(f"Match: {message == decrypted_message}")
